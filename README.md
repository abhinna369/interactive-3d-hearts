# 💖 Interactive 3D Hearts Experience

An immersive and interactive 3D visual experience built with **Three.js** and **WebGL**.

---

## ✨ Overview

This project transforms a simple web page into a magical interactive world filled with floating glowing hearts, sparkling fairy dust, cinematic lighting, and hidden love messages.

Every heart gently drifts through space. Clicking on one creates a magical burst of particles, a glowing shockwave, and reveals a random heartfelt message.

Designed entirely with vanilla JavaScript and Three.js, the project showcases modern WebGL techniques, particle systems, procedural textures, post-processing effects, and interactive animation.

---

## ✨ Features

- ❤️ Floating 3D hearts
- ✨ Magical particle explosions
- 🌌 Nebula-inspired heart background
- 💬 Hidden love messages
- 🌟 Bloom lighting
- 🎥 Smooth camera motion
- 🖱️ Interactive mouse trail
- ⚡ Pure JavaScript
- 🎮 Built using Three.js
